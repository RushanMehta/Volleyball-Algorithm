How the Math and Predictions Work

This system models game progressions step-by-step using Dynamic Programming with Rally Scoring Rules.

1. The Core Equation

At any score, the model computes the odds of winning the set by choosing the serve action that yields the highest mathematical probability of success.

Explicit Boundary & Modeling Simplifications (Academic Disclaimers)
1. First-Order Rally Approximation:
Real match play has infinite back-and-forth options (Serve, Pass, Set, Attack, Dig, Block, etc.). For mathematical simplicity, this system condenses the rally into the immediate pass quality and the opponent's average offensive success off that pass. This represents a highly effective first-order projection.
2. Constant Opponent Defense:
This model assumes opponent passing and attack rates remain steady. It does not factor in opponent adjustments, fatigue, or rotation changes.
3. Optimization Boundary:
The outputs are optimized under these specific model constraints. Real-world coach intuition and player confidence should always supplement these computer recommendations.
