Mathematical Formulation & Framework

This computational framework models volleyball sets as a finite-horizon Markov Chain resolved via an iterative bottom-up dynamic programming matrix calculation sequence.

Equation 1: Rally Win Probability Decomposition ($P_{\text{rally}}$)

$$P(\text{Win Rally} \mid \text{Serve}) = P(\text{Ace}) + \sum_{s \in \text{States}} P(\text{Outcome}_s) \times (1.0 - \text{Kill}_s)$$

Equation 2: Expected Points Per Serve (EPPS) Mapping

$$\text{EPPS} = P(\text{Ace}) + \sum_{s \in \text{States}} P(\text{Outcome}_s)(1.0 - 2\cdot\text{Kill}_s) - P(\text{Error})$$

Equation 3: Statistical Variance Propagation Boundary (Wald Interval)

$$\text{Margin of Error (MOE)} = Z_{0.95} \times \sqrt{\frac{P_{\text{rally}}(1 - P_{\text{rally}})}{n}}$$

Strategic Model Simplifications & Compressed Chains
Acknowledged Simplification Flow:
Instead of modeling highly dimensional variables across every sub-contact sequence:
$$\text{Serve Choice} \longrightarrow \text{Reception} \longrightarrow \text{Setter Location} \longrightarrow \text{Block System} \longrightarrow \text{Transition Defense} \longrightarrow \text{Outcome}$$ The system maps the direct outcome probability distributions directly into an aggregated defensive kill efficiency parameter ($\text{Kill}_s$), condensing secondary handling adjustments into the state ruleset.
