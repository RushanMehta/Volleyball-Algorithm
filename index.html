Mathematical Formulation & Policy Derivation

This application treats a competitive volleyball set as a Single-Agent Markov Decision Process (MDP) with a Stationary Opponent Policy under standard Rally Scoring Rules.

1. The Bellman Optimality Equation

The solver calculates the globally optimized set win equity at every possible score state by selecting the action maximizing the downstream transition value:

$$V(u, t, \text{serve}) = \max_{a \in \{\text{TS}, \text{FL}\}} \sum_{u', t', s'} P(u', t', s' \mid u, t, \text{serve}, a) \cdot V(u', t', s')$$

Academic & Structural Disclosures
1. Semi-Markov Structural Compression Approximation:
Real match play propagates across multi-contact sequence variations (Serve $\rightarrow$ Pass $\rightarrow$ Set $\rightarrow$ Attack $\rightarrow$ Dig). This framework uses a high-reliability semi-Markov approximation, compressing intermediate contacts into direct opponent defensive kill capabilities ($\text{Kill}_s$) mapped from entry pass execution.
2. Stationary Strategy Limitation:
This model optimizes a single-agent decision tree against a fixed, stationary opponent policy profile (represented by a static Sideout Rate). In tournament play, opponent response behaviors are dynamic (a two-player stochastic game).
3. Chain Truncation Bounds:
An exact absorbing Markov chain handling infinite deuce extensions contains unlimited dimensions. To remain computationally efficient, this application uses a boundary truncation technique at a structural score cap, resolving tied horizons cleanly without terminal drift.
